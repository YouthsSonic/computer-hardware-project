Hardware Terminology

Input Device

An input device is an device used to give a computer signals. Some examples of an input device is an keyboard, mouse or even a pen.

Output Device

An output device is a peice of computer hardware that converts 1s and 0s into informaation humans can read. Some examples of an input device is a colored display or monitor.

A central proccesing unit which is also known as a CPU is a peice of computer hardware that carries out instructions given to it.

Arethmetic Logic Unit

An arethemetic logic unit which is also known as an ALU is an digital that is typically used to do aresthmetic fuctions.

Control Unit

An control unit also known as a CU is a part in a computers CPU. The CU manages the operation of the CPUS processor. It also tells it how to react to instructions that were sent to the CPU.

Random Access Memory

Random acess memory is a short term type of computer memory. Random access memory is also known as RAM for short. RAM usually works with the computers direct access memory.

Optical Memory

Optical storage stores data on a physical disk by making marks in it so the computer can read it. Optical Memory looks a lot like an CD.

Magnetic Memory

Magnetic memory uses magnets to store data on a computer. It uses multiple patterns of magntisation to store data.

Logic gates are the foundation of all digital systems.Logic gates are an electronic circuits that have one or more than one input.
